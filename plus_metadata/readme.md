# plus_metadata
It is a script that write metadata to PNG file. That metadata is recognized in CivitAi.
## requirements
python modules
```
pip install pillow FreeSimpleGUI pyperclip
```
## How to use
1. Run this script.
2. Input parameters.
3. Click run button.
4. Files that ".png" of PNG files turn into "_meta.png" is made.
