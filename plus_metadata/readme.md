# plus_metadata
It is a script that write metadata to PNG file and JPG file. That metadata is recognized in CivitAi.
## requirements
python modules
```
pip install pillow FreeSimpleGUI pyperclip pyexiv2
```
## How to use
1. Run this script.
2. Input parameters.
3. Click run button.
4. In case of PNG file, files that ".png" of PNG files turn into "_meta.png" is made.
   In case of JPG file, files that ".jpg" of JPG files turn into "_meta.jpg" is made.
